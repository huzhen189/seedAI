### 记忆四层架构
- **L0** MySQL Message 完整对话
- **L1** Redis `summary:{conv_id}` 滚动摘要，1d TTL，过期 MySQL 重压
- **L2** done→LLM 精炼→覆写 Message.content→refined SSE 事件
- **L2+** 蒸馏：done→LLM 抽取结构化条目→project_memories 表 + Chroma
- **L3** Chroma user_preferences + User.card_json

### Chroma 六集合
| 集合 | 隔离 key | 用途 |
|------|----------|------|
| `conversation_context` | conversation_id | 对话上下文相似度 |
| `user_preferences` | user_id | 用户风格/约束偏好 |
| `project_memory` | project_id | 项目决策/需求历史 |
| `project_code` | project_id | 大项目代码语义索引 |
| `error_patterns` | 全局 | 错误模式库（20条种子） |
| `components` | 无 | 公共组件库 |
